# Tag Game Web AI

A mobile-friendly single-player tag game. Tap or click to move, avoid the red "It" player, and tag bots when you're It.

## Features

- Tap/click movement and WASD keyboard movement
- Bot chase and flee behavior
- Score and round timer drawn inside the arena map
- Tag shake and mobile vibration feedback
- Untagged players speed up after each tag, then smoothly slow back down

## Run locally

```bash
npm install
npm run dev
```

Open the local Vite URL shown in your terminal, usually `http://localhost:5173/`.
